thunder.SVD¶

class thunder.SVD(k=3, method='auto', maxIter=20, tol=1e-05)¶

Singular value decomposiiton on a distributed matrix.

Parameters:

k : int, optional, default = 3

Number of singular vectors to estimate

method : string, optional, default “auto”

Whether to use a direct or iterative method. If set to ‘auto’, will select preferred method based on dimensionality.

maxIter : int, optional, default = 20

Maximum number of iterations if using an iterative method

tol : float, optional, default = 0.00001

Tolerance for convergence of iterative algorithm

Attributes

u (RowMatrix, nrows, each of shape (k,)) Left singular vectors
s (array, shape(nrows,)) Singular values
v (array, shape (k, ncols)) Right singular vectors
